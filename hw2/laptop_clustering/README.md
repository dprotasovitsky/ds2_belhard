## Анализ кластеризации ноутбуков

Проект для анализа и кластеризации данных о ноутбуках с использованием различных алгоритмов машинного обучения.
### О проекте

Этот проект предоставляет полный пайплайн для анализа датасета ноутбуков, включая предобработку данных, исследовательский анализ, кластеризацию и визуализацию результатов. Используются современные методы машинного обучения для выявления сегментов ноутбуков на рынке.
### Основные возможности

   * Предобработка данных: автоматическая обработка пропущенных значений, кодирование категориальных признаков

   * Разведочный анализ: визуализация распределений, корреляционный анализ

   * Кластеризация: несколько алгоритмов (иерархическая, DBSCAN)

   * Понижение размерности: UMAP, LDA, PCA для визуализации и улучшения кластеризации

   * Оценка качества: метрики Silhouette, Calinski-Harabasz, Davies-Bouldin

   * Логирование: детальное протоколирование всех этапов анализа

   * Визуализация: автоматическая генерация графиков и диаграмм

### Запуск анализа

1. Поместите файл с данными laptopCleanData.csv в корневую директорию проекта

2. Запустите главный скрипт:

        bash

        python main.py

### Методы кластеризации
#### Реализованные алгоритмы

   1. Иерархическая кластеризация:

       * Метод: Ward

       * Визуализация: дендрограмма

       * Автоматическое определение числа кластеров

    2. DBSCAN:

        * Автоматический подбор параметра eps

        * Запасной вариант с KMeans при неудачной кластеризации

        * Обработка шумовых точек

        * Применение на разных представлениях данных (оригинальные, UMAP, PCA)


### Методы понижения размерности

    * UMAP: сохранение локальной структуры данных

    * LDA: линейный дискриминантный анализ с использованием ценовых категорий

    * PCA: метод главных компонент

### Метрики оценки

Проект использует следующие метрики для оценки качества кластеризации:

    * Silhouette Score: мера схожести объектов внутри кластеров

    * Calinski-Harabasz Index: отношение межкластерной дисперсии к внутрикластерной

    * Davies-Bouldin Index: среднее сходство между кластерами

### Входные данные
Данные получены с kaggle: https://www.kaggle.com/datasets/shahriarkabir/laptop-dataset-clean/data

#### О наборе данных
Здесь приведены описания столбцов:

 1. brand_name: Название бренда или производителя ноутбука.
 2. name: Название модели или идентификатор ноутбука.
 3. processor_type: тип процессора, используемого в ноутбуке (например, 4. Intel Core i5, AMD Ryzen 7, Apple M1).
 4. processor_brand: марка процессора (например, Intel, AMD, Apple).
 5. generationsя: Поколение процессора (например, 11-е, 12-е, 13-е).
 6. ram: объем оперативной памяти (ОЗУ с произвольным 6.доступом) в гигабайтах (ГБ), установленный в ноутбуке.
 7. storage_capacity_gb: Общая емкость устройств хранения данных ноутбука в гигабайтах (ГБ).
 8. have_ssd: Логическое значение, указывающее, установлен ли в ноутбуке твердотельный накопитель (SSD) или нет.
 9. have_hdd: Логическое значение, указывающее, есть ли в ноутбуке жесткий диск (HDD) или нет.
 10. graphics_capacity: Графический объем или выделенная графическая память в гигабайтах (ГБ), если применимо.
 11. цена: Стоимость ноутбука в указанной валюте.
 12. display_size_inch: Размер экрана ноутбука в дюймах.
13. display_type: Тип дисплея, используемого в ноутбуке (например, FHD, OLED, Liquid Retina).
14. color: Цвет ноутбука.

### Настройка параметров

Основные параметры можно настроить в config/settings.py:

    python

    # Параметры кластеризации
    DEFAULT_N_CLUSTERS = 4           # Число кластеров для иерархической кластеризации
    DBSCAN_MIN_SAMPLES = 5           # Минимальное число samples для DBSCAN
    UMAP_NEIGHBORS = 15              # Число соседей для UMAP
    UMAP_MIN_DIST = 0.1              # Минимальное расстояние для UMAP

    # Настройки логирования
    LOG_LEVEL = logging.INFO          # Уровень детализации логов

### Интерпретация результатов

Проект автоматически интерпретирует кластеры, предоставляя:

    * Статистику по каждому кластеру (цена, RAM, память, графика)

    * Определение ценового сегмента (бюджетный, средний, премиум, люкс)

    * Топ-3 бренда и процессора в каждом кластере

    * Процент игровых ноутбуков
